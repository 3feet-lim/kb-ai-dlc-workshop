# User Personas

## Persona 1: 고객 (Customer)

| Attribute | Description |
|-----------|-------------|
| Name | 김민수 (대표 고객) |
| Role | 매장 방문 고객 |
| Device | 테이블 태블릿 (브라우저 기반) |
| Goal | 대기 없이 빠르고 편리하게 음식 주문 |
| Pain Points | 직원 호출 대기, 메뉴판 가독성, 주문 누락 |
| Tech Comfort | 보통 (스마트폰 사용 수준) |
| Key Needs | 직관적 UI, 빠른 메뉴 탐색, 주문 상태 확인 |

**사용 시나리오**: 매장에 입장하여 테이블에 앉으면 태블릿에서 메뉴를 탐색하고, 장바구니에 담은 후 주문을 확정한다. 주문 후 상태를 실시간으로 확인한다.

## Persona 2: 매장 관리자 (Store Admin)

| Attribute | Description |
|-----------|-------------|
| Name | 박지영 (대표 관리자) |
| Role | 매장 운영자/점주 |
| Device | PC 또는 태블릿 (브라우저 기반) |
| Goal | 효율적인 주문 관리 및 매장 운영 |
| Pain Points | 주문 누락, 수기 관리 비효율, 테이블 상태 파악 어려움 |
| Tech Comfort | 보통~높음 |
| Key Needs | 실시간 주문 모니터링, 메뉴 관리, 테이블 세션 관리 |

**사용 시나리오**: 매장 오픈 시 관리자 화면에 로그인하여 실시간으로 들어오는 주문을 모니터링하고, 주문 상태를 변경하며, 고객 퇴장 시 테이블 이용 완료 처리를 한다.
